# most_occured_symbol
**prerequisite:**  
python3  
python3 Module virtualenv

**how to install:**  
```
~ git clone https://github.com/AntzInThePantz/most_occurred_symbol.git  
~ cd most_occurred_symbol  
~ python3 -m venv .  
~ . bin/activate  
~ python3 setup.py install  
```

**how to execute:**  
```
~ most-occurrences
```
